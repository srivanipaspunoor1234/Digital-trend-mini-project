# Digital Trends Website

A responsive and modern **Digital Marketing Agency Website** built using  
**HTML, CSS, Bootstrap, and Font Awesome**.

This project showcases a clean UI design with carousel, testimonials, and a professional footer layout.

Features

- Responsive navigation bar
- Hero section with call-to-action button
- Bootstrap carousel for digital trends
- Client testimonials section
- Fully styled footer with contact details
- Smooth hover effects and animations

Technologies Used

- HTML5  
- CSS
- Bootstrap 5  
- Font Awesome
